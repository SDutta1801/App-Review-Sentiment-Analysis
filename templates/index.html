APP REVIEW SENTIMENT ANALYSIS

Our Innovation

This App basically deals with the problems every app user faces while downloading apps on any Google or Apple Play Store. When we type any app name we read a few comments but an App contains a lot of comments sitting, reading and then analysing the app whether it is good / average / below average is tough. So, we came up with this idea in which if we give any App name on the search box it will do a Sentiment Analysis on all of its reviews and give a feedback on whether it is Good/Average/Below Average.
